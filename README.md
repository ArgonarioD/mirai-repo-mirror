# Mirai Repo

用于放置包版本信息。

## 如何在此仓库被列出

1. 包**必须遵循**`AGPLv3`协议开源
1. `JAR`必须已在`Maven Central`或`JCenter`上线（暂定，之后会支持更多途径）
1. 提交`Pull Request`
